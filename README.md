# Interactive-Stopwatch-Web-Application

## Overview

This project is a responsive and interactive stopwatch web application developed using HTML, CSS, and JavaScript. It provides accurate time tracking with milliseconds, lap recording functionality, keyboard shortcuts, and a modern glassmorphism user interface.

## Features

* Start, Pause, and Reset stopwatch
* Millisecond precision timing
* Lap time recording
* Delete individual lap records
* Real-time date and clock display
* Keyboard shortcuts:

  * Space → Start/Pause
  * L → Record Lap
  * R → Reset
* Responsive design
* Modern animated UI

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Project Structure

```
Interactive-Stopwatch/
│
├── index.html
├── style.css
└── script.js
```

## How to Run

1. Download or clone the repository.
2. Open `index.html` in any modern web browser.
3. Start using the stopwatch.
